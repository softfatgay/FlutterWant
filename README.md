## 主要功能点
- 网络请求封装，返回数据实例化模型
- Flutter与原生Android/iOS的交互
- 常见的安卓Material Design风格
- 状态管理，局部刷新
- 纯属学习项目，数据全部来源于网易严选Web端,切勿涉及违法行为
- 数据来之不易，如有帮助，请帮忙点个👍
